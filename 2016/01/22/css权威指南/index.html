后代选择器包含子、孙。。。所有子子孙孙内容；
子选择器只包括子的内容，不包括孙的内容；

width: 300px;
}

background: orange;
}
body>div{
background: green;

结构伪类选择器

:nth-child(xx)
xx有三部分组成，“－4n＋5”，其中－代表方向，＋往右，－往左；4代表每次循环总位数，＋5代表从第5位往右开始循环；
